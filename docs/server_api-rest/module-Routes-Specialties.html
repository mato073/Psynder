(route) DELETE /specialties/:id

Route:
MethodPath
DELETE /specialties/:id

Route to add a specialty to the list of specialties.

Authentication

Need a variable Authorization in the header with value Bearer <accessToken>.

If authentication fails it will return a 401 error.

Header Parameters:
Name Type Description
Authorization String Refresh token of the user
Route Parameters:
Name Type Description
:id String

ID of the specialty you want to delete
Response:
Name Type Description
json JSON Response as JSON type
fail_json.message String Message to know why the opration failed
Response Code:
Type Description
200 If the request is successful
400 If the request failed
401 If authentication fails
