(constant) AUTHSOURCE :string

Global that stores the auth source of the database the user is created in. By default it is "admin".

Type:
  • string
Default Value:
  • "admin"

(constant) DBNAME :string

Global that stores the name of the database we're working with.

Type:
  • string

(constant) DBURI :string

Global that stores the URI needed by MongoDB to connect to a database with a username, password, host, port, database's name and auth.

It should look like that mongodb://<username>:<password>@<host>:<port>/<database's name>?authSource=<auth source>

Type:
  • string
Default Value:
  • "admin"

(route) GET /emails/is-password-reset-token-valid

Route:
MethodPath
GET /emails/is-password-reset-token-valid

Route to verify password reset token provided in email received via POST /reset-password

Query Parameters:
Name Type Description
token String Password reset token provided in the password reset email
Response:
Name Type Description
json JSON Response as JSON type
json.message String Message to know if the operation succeeded or failed
Response Code:
Type Description
200 if the request is successful
400 if the request failed
401 if the request is unauthorized and the token is therefore invalid
500 if there was an internal server error
503 if the database got disconnected

(constant) HOST :string

Global that stores the host of our database. By default it is "localhost"

/!\ Warning : If the database is stored in a local docker container, it may be the name of the container not "localhost" /!\

Type:
  • string
Default Value:
  • "localhost"

(constant) HOST :string

Global that stores the host of our redis cache. By default it is "localhost"

/!\ Warning : If the redis cache runs in a local docker container, it may be the name of the container not "localhost" /!\

Type:
  • string
Default Value:
  • "localhost"

(constant) PORT :string

Global that stores the port of our database. By default it is "27017"

/!\ Warning : If the database is stored in a local docker container, it may be the port you linked to /!\

Type:
  • string
Default Value:
  • "27017"

(route) POST /emails/reset-password

Route:
MethodPath
POST /emails/reset-password

Route to request a password reset email

Body Parameters:
Name Type Description
email String Email of the user or therapist
redirectTo String Route you the user redirected to when he clicks on the link provided in the email. Ex: http://psynder.fr/auth/reset-password. The domain the user is redirected to is automatically deciphered by the server when request sent through web client.
isTherapist Boolean Specifies if the request is supposed to be sent by a therapist or not
Response:
Name Type Description
json JSON Response as JSON type
json.message String Message to know if the opration succeed or failed
Response Code:
Type Description
200 if the request is successful
400 if the request failed
500 if there was an internal server error
503 if the database got disconnected

(route) POST /token/refresh

Route:
MethodPath
POST /token/refresh

Route to refresh an authenticated session (user or therapist). Both access AND refresh Tokens are refreshed, so it goes for their expiration date as well.

Body Parameters:
Name Type Description
refreshToken String Refresh token provided earlier to the user or therapist
Response:
Name Type Description
json JSON Response as JSON type
fail_json.message String Message to know if the opration succeed or failed
success_json.accessToken String New accessToken
success_json.refreshToken String New refreshToken
Response Code:
Type Description
200 if the request is successful
400 if the request failed
