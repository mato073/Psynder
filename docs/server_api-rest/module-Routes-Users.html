Route:
MethodPath
DELETE /users/current

Route to delete the current connected user.

/!\ Needs access token ! /!\

Authentication

Need a variable Authorization in the header with value Bearer <accessToken>.

If authentication fails it will return a 401 error.

Header Parameters:
Name Type Description
Authorization String Refresh token of the user
Response:
Name Type Description
json JSON Response as JSON type
json.message String Message to know if the opration succeed or failed
Response Code:
Type Description
200 If the request is successful
400 If the request failed
401 If authentication fails

(route) GET /users/current/messages?nlasts=0

Route:
MethodPath
GET /users/current/messages?nlasts

Route to get messages between a therapist and the non-therapist user

Authentication

Need a variable Authorization in the header with value Bearer <accessToken>.

Header Parameters:
Name Type Description
Authorization String Refresh token of the user
Query Parameters:
Name Type Attributes Default Description
nLasts String,Int optional 0 (Optional) If specified and number is superior to 0, will return the N last messages, else all messages will be returned
Route Parameters:
Name Type Description
:receiverId String

ID of the therapist you want to fetch messages with
Response:
Name Type Description
json JSON Response as JSON type
json_fail.message String Message to know if the operation succeed or failed
json_success.messages String Array of messages
json_success.messages[i]._id String ID of the message instance
json_success.messages[i].sender String String to know who sent the message, should be 'therapist' or 'user'
json_success.messages[i].date String Date of the message
json_success.messages[i].content String Content of the message
Response Code:
Type Description
200 If the request is successful
400 If the request failed
401 If the authentification fails
