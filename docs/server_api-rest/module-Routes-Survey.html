(route) POST /survey/users/results

Route:
MethodPath
POST /survey/users/results

Route to send the results of the algorithm of detection of potential depression disorders.

/!\ Needs access token ! /!\

Authentication

Need a variable Authorization in the header with value Bearer <accessToken>.

If authentication fails it will return a 401 error.

Body Parameters:
Name Type Description
body_json JSON Results of the algorithm of detection of depression disorders (**object as string (may use JSON.stringify)**)
body_json.potentialDisorder String Name (in french) of the disorder managed by the file
body_json.Questions Array.<Object> Questions the had to answer
body_json.Questions[i].name String May be a ID of question
body_json.Questions[i].question String The question asked to the user
body_json.Questions[i].types Array.<String> Types waited for question asked to the user. Could be one (or many) of the following: "BOOL", "ANSWERS", "DATE.NEAR", "DATE.FAR", "BRUT.INT", "BRUT.STRING"
body_json.Questions[i].answered Bool To know if the question answered by the user
body_json.Questions[i].value String The answer to the question asked to the user
body_json.Questions[i].answers Array.<Object> (Optionnal) Possible answers to the question asked to the user
body_json.Questions[i].answers[j].name String Value (answer) of the possible answers to the question asked to the user
body_json.Questions[i].answers[j].value Number Value (for score) of the possible answers to the question asked to the user
body_json.Questions[i].forksAnswers Array.<Object> (Optional) Forks of values of a brut answer (as number)
body_json.Questions[i].forksAnswers[j].fork String The fork of the fork answers to the question asked to the user. Has to be like "value1**-**value2" or "value**+**"
body_json.Questions[i].forksAnswers[j].value Number Value (for score) of the fork answers to the question asked to the user
Header Parameters:
Name Type Description
Authorization String Refresh token of the user
Response:
Name Type Description
json JSON Response as JSON type
json.message String Message to know if the opration succeed or failed
Response Code:
Type Description
200 If the request is successful
400 If the request failed
401 If authentication fails
