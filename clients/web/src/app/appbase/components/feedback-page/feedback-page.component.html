Donner son avis

Vous pouvez nous faire un retour d'expérience sur ce que vous avez aimé ou non, que vous soyer thérapeute ou non.
Vous pouvez aussi nous faire suggestions pour améliorations, le tout en cliquant sur le bouton ci-dessous.

Tout type de retour est le bienvenu et il le sera encore plus si vous pouvez apporter un maximum de détails !

Contacter le support

Si vous avez rencontré un problème technique (bouton défectueux, lien invalide, etc) ou même un problème avec un autre utilisateur, vous pouvez contacter notre support en cliquant sur le bouton ci-dessous.

En cas de nécessité nous vous recontacterons à l'adresse mail précisée lors de la création de votre ticket.
Il est donc important de bien remplir tous les champs correctement.
